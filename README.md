# 🌊 Selamat datang di Repo Github SIGAP DESA GMLS! 🌊

SIGAP DESA adalah website yang digunakan untuk menyimpan data penduduk di Lebak Selatan untuk keperluan mitigasi bencana. Website ini dibuat karena data Pemdes tidak cukup untuk keperluan mitigasi. Maka dari itu, website ini tidak hanya menyimpan informasi biodata penduduk, namun juga menyimpan informasi seperti bahan konstruksi rumah, kerawanan bencana dari rumah seorang penduduk, titik koordinat rumah penduduk, dll. 

## Cara berkontribusi
Bagi mahasiswa FTI yang mengikuti Humanity Project dan ditugaskan untuk mengerjakan Sigap Desa, silahkan pelajari cara menggunakan Github dan Git terlebih dahulu agar bisa berkontribusi. Setelah Anda sudah mempelajari cara menggunakan Github dan Git, silahkan kontribusi ke repo ini dengan mengikuti arahan yang ada di website [ini](https://docs.github.com/en/get-started/exploring-projects-on-github/contributing-to-a-project).
